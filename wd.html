Week 2 : Introduction to HTML, CSS and more on the micro:bit

23.01.2020

Today's class was a session of confusion for me. We learned how to create our own webpage with HTML and CSS. Everyone seemed to be ahead of me and I felt like I was annoying the people around me with my anxiousness and questions. I need to calm down. I think this was because the things we did were step-by-step progressive things, so since I missed the first few steps, I could only struggle in stagnation while others moved ahead. It's a terrible feeling.

However, I am happy to say that I managed to clarify all my doubts and create a webpage by myself! Going through Yuta's step-by-step document again (with the luxury of time) made things clear! Althought I spent a whole day++ doing this, I am so glad I did. And once I was able to accomplish the basics that were taught in class, I began to seek other ways to improve my webpage. Over the weekend, I learned how to -

  • Add hyperlinks (for pages within and outside this webpage);
  • Create hyperlinks that open in a new tab (instead of redirecting);
  • Create links that allow readers to scroll straight to a specific line on the page;
  • Create margins (this improved the look of the webpage significantly);
  • Add a horizontal line (for segmantation);
  • Break a line (like "enter", except that html ignores and "enters" made here and the words just clump up);
  • Add emojis and gifs.

We also continued learning with the micro:bit, with the inclusion of electrical components. I was so lucky to have had some experience with electrical components in Ngee Ann, or I would not have been so calm in assembling the electrical arrangement. Together with Kianne, I was able to -

  • Make an LED blink;
  • Control button inputs;
  • Adjust a variable resistor.

The last one was a little tricky because the adjustable switch was a little different from the diagram, so making it work required some technical knowledge.

LEARNING TAKEAWAYS

  • Creating my very own webpage with HTML and CSS.

HOMEWORK

  • Create own webpage for my weekly documentation (aka this page!!).
